# simple_tcp_server_cs
a csharp implementation of simple_tcp_server.

This project is a C# compatible version of https://github.com/GGN-2015/simple_tcp_server.

## Sample Usage

See [./src/SampleUsage.cs](./src/SampleUsage.cs)

## Run Sample Usage

> [!IMPORTANT]
> Make sure you have `dotnet` on your `PATH` before run `LaunchTest.bat`

```cmd
src\LaunchTest.bat
```

## In Unity

To use `SimpleTcpClient` in Unity, you need to copy `src\SimpleTcpClient.cs` and `src\Utils.cs` to your Unity project.

The project has been tested under Unity Edtior version:
- `2022.3.62f3c1 (LTS)`
